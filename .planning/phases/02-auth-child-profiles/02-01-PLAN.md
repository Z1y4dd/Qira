- **Task 1.4 — Author `requireParent()` real body with lazy parent-row upsert**
  - **Files:** `src/services/profiles.ts`
  - **Action:**
    - Replace the `not implemented until Phase 2` body of `requireParent()` with the real implementation:
      1. Read cookies via the `cookies()` helper imported at the boundary of the calling Server Action (NOT inside the Service — the Service is framework-agnostic per Phase 1 invariant). The Service takes a `SupabaseClient` instance as an injected argument: `export function requireParent(supabase: SupabaseClient): Promise<ParentSession>`.
      2. Call `supabase.auth.getUser()` (NEVER `getSession`).
      3. If `data.user === null`, throw a typed `AuthError('UNAUTHENTICATED')`.
      4. If `data.user.email_confirmed_at === null`, throw `AuthError('UNVERIFIED')`.
      5. Lazy-upsert into `parents`: `await db.insert(parents).values({ id: brandedParentId(data.user.id), email: data.user.email! }).onConflictDoNothing({ target: parents.id })`.
      6. Return `{ parentId: brandedParentId(data.user.id), email: data.user.email! }`.
    - Define `AuthError` in the same file: `export class AuthError extends Error { constructor(public reason: 'UNAUTHENTICATED' | 'UNVERIFIED' | 'NO_ACTIVE_CHILD') { super(reason); } }`.
    - Add `brandedParentId(id: string): ParentId` helper (just `return id as ParentId`).
    - **Critical:** do NOT import from `next/*` in this file. The Service Layer Purity test will fail if you do.
  - **Verification:** `pnpm tsc --noEmit` clean. `tests/invariants/service-layer-purity.test.ts` passes (greps for `from 'next/`). Manually trace: caller (Server Action) imports `cookies` from `next/headers`, passes the result to `createClient` from `@/utils/supabase/server`, passes the client to `requireParent`. Service stays pure.
  - **Commit:** `feat(services/profiles): implement requireParent with getUser + email-verified check + lazy parent upsert`

- **Task 1.5 — Wire the hard email-verification gate into `(authenticated)/layout.tsx`**
  - **Files:** `src/app/(authenticated)/layout.tsx`
  - **Action:**

    - Body:
      ```ts
      const supabase = createClient(await cookies());
      try {
        await requireParent(supabase);
      } catch (err) {
        if (err instanceof AuthError && err.reason === 'UNAUTHENTICATED') redirect('/sign-in');
        if (err instanceof AuthError && err.reason === 'UNVERIFIED') redirect('/verify-email');
        throw err;
      }
      return <>{children}</>;
      ```
