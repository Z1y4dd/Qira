**Function implementations:**

1. `getNextTextForChild(childId: ChildId): Promise<import('./library').LeveledText | null>`

   - Load the child's `currentLevelId` from `childProfiles` WHERE `id = childId`.
   - If `currentLevelId` is null, throw `new AuthError('NO_ACTIVE_CHILD')` — child has no assigned level (unplaced).
   - Find all text IDs the child has already FINISHED reading: `SELECT DISTINCT textId FROM attempts WHERE childId = childId AND kind = 'reading' AND finishedAt IS NOT NULL`. Use `isNull` negated — Drizzle: `isNull(attempts.finishedAt)` is the helper; for NOT NULL use `sql\`${attempts.finishedAt} IS NOT NULL\``.
   - Query `texts` WHERE `levelId = child.currentLevelId` AND `id NOT IN (finishedTextIds)` ORDER BY `texts.createdAt ASC, texts.id ASC`. Use `inArray` for the exclusion list. If `finishedTextIds` is empty, skip the `NOT IN` filter.
   - Fallback: if all texts at this level are exhausted (query returns zero rows), re-query WITHOUT the NOT IN filter (return first text: `ORDER BY createdAt ASC LIMIT 1`). This implements the "repeat if all attempted" rule (pre-resolved decision #8).
   - Look up `levelRow.number` from `levels` WHERE `id = text.levelId`.
   - Return a `LeveledText` shape: `{ id, level: levelRow.number, titleAr: text.titleAr, bodyAr: text.bodyAr, tashkeelDefault: levelRow.number <= 10 ? 'on' : 'off', illustrationUrl: null }`. Import `LeveledText` from `./library` — that type already exists.

2. `getTextWithQuestions(textId: TextId, attemptId: AttemptId): Promise<{ text: import('./library').LeveledText, questions: ComprehensionQuestion[] }>`

   - Query `texts` WHERE `id = textId` with level join.
   - Query `questions` WHERE `textId = textId AND kind = 'comprehension'` ORDER BY `position ASC`.
   - If no questions found, throw `new Error(\`getTextWithQuestions: no comprehension questions for text ${textId}\`)`.
   - Fetch all choices: `choices` WHERE `questionId IN (questionIds)`. Use `inArray`.
   - For each question, apply `deterministicShuffle` to its choices using seed `${attemptId}:${question.id}` (pre-resolved decision #12). Strip `isCorrect` from the returned choices — return only `{ id: ChoiceId, labelAr: string }`.
   - Return `{ text: LeveledText, questions: ComprehensionQuestion[] }`.

3. `startComprehensionAttempt({ childId, textId }: { childId: ChildId, textId: TextId }): Promise<AttemptId>`

   - Parse with `StartComprehensionAttemptInput`.
   - Verify the child has a `currentLevelId` (requiresActiveChild is already called in the Server Action; this is defense in depth — if null, throw).
   - Insert into `attempts`: `{ childId, kind: 'reading', textId }`. Do NOT set `placementBankVersion` — that field is for placement attempts only. All other nullable columns default to null.
   - Return the inserted `row.id` cast to `AttemptId` (pattern: `placement.ts:219-222`).

4. `recordComprehensionAnswer(input: RecordComprehensionAnswerInput): Promise<RecordComprehensionAnswerResult>`

   This is the most complex function — mirror `recordPlacementAnswer` (placement.ts:315-451) with comprehension-specific completion logic.

   - Parse with `RecordComprehensionAnswerInput` Zod schema.
   - Load the attempt: SELECT `id, childId, textId, finishedAt` FROM `attempts` WHERE `id = attemptId`. If not found or `childId !== input.childId`, throw `new AuthError('NO_ACTIVE_CHILD')` (cross-child defense).
   - If `finishedAt` is already set, throw `new Error('ATTEMPT_ALREADY_FINISHED')`.
   - Look up `isCorrect` from `choices` WHERE `id = chosenChoiceId AND questionId = questionId`. If not found, throw. This is the server-authoritative scoring — `isCorrect` is read here and NEVER returned to the client.
   - Re-derive the choice order that was displayed to the child: fetch all choices for `questionId`, apply `deterministicShuffle(choices, \`${attemptId}:${questionId}\`)`. Store this as JSONB `choiceOrder` — pattern: `placement.ts:360-361`.
   - Insert into `attempt_answers`: `{ attemptId, questionId, chosenChoiceId, isCorrect: choice.isCorrect, choiceOrder: shuffledChoiceIds }`. `answeredAt` defaults to `NOW()`.
   - Count total comprehension questions for this text: SELECT COUNT(*) FROM `questions` WHERE `textId = attempt.textId AND kind = 'comprehension'`.
   - Count answers recorded so far for this attempt: SELECT COUNT(*) FROM `attempt_answers` WHERE `attemptId = attemptId`. (The row just inserted counts.)
   - If answers count < total questions: return `{ correct: choice.isCorrect === 1, nextQuestionIndex: answersCount + 1, finalResult: null }`.
   - If answers count == total questions (all answered): finalize the attempt.
     - Count correct answers: SELECT COUNT(*) FROM `attempt_answers` WHERE `attemptId = attemptId AND isCorrect = 1`.
     - UPDATE `attempts` SET `finishedAt = NOW(), score = correctCount` WHERE `id = attemptId`.
     - Load text title for the result: SELECT `titleAr` FROM `texts` WHERE `id = attempt.textId`.
     - Load level number: SELECT `number` FROM `levels` JOIN `texts` ON `texts.levelId = levels.id` WHERE `texts.id = attempt.textId`.
     - Return `{ correct: choice.isCorrect === 1, nextQuestionIndex: null, finalResult: { attemptId, score: correctCount, totalQuestions: total, textTitleAr: text.titleAr, level: levelNumber } }`.

5. `getReadingAttemptResult(attemptId: AttemptId): Promise<ReadingAttemptResult>`

   - Load attempt: SELECT `finishedAt, score, textId, childId` FROM `attempts` WHERE `id = attemptId AND kind = 'reading'`.
   - If not found or not finished, throw `new Error('ATTEMPT_NOT_FINISHED')`.
   - Load text: SELECT `titleAr, levelId` FROM `texts` WHERE `id = attempt.textId`.
   - Load level number: SELECT `number` FROM `levels` WHERE `id = text.levelId`.
   - Count total questions: SELECT COUNT(*) FROM `questions` WHERE `textId = attempt.textId AND kind = 'comprehension'`.
   - Return `ReadingAttemptResult` shape.
