# Plan 01: Repo + RTL Bootstrap

## Goal

Stand up the Next.js 16 + Tailwind v4 + shadcn `--rtl` + `next-intl` repo scaffold with a single Arabic landing page at `/ar/` that renders RTL by default with Noto Naskh body + Cairo UI fonts self-hosted via `next/font/google`. Defines the route shape every later slice plugs into.

## Demo gate

Running `pnpm dev` and visiting `http://localhost:3000/` redirects to `http://localhost:3000/ar/`. DevTools inspector shows `<html lang="ar" dir="rtl">`, body computed `font-family` matches `Noto Naskh Arabic`, and Network tab shows the font files served from `/_next/static/media/*.woff2` (same-origin) with zero requests to `fonts.googleapis.com` or `fonts.gstatic.com`. The page displays the literal `مرحباً بكم في قِراءة` (Phase 1 step — this exact literal will be re-routed through `<ArabicText>` in Slice 4). A shadcn `<Button>` underneath the heading renders with kid-appropriate sizing.
